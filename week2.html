Sprint Retrospective

Week 2 Sprint Retrospective

What went well?

I conducted user interviews and extracted useful insights that helped identify core pain points around food storage, expiry confusion, and waste. I felt engaged during analysis and was able to cluster key themes quickly. Creating the empathy map and persona clarified the user perspective. I stayed on track with my Sprint goal of developing user understanding and generating insights.

What didn't go well?

Some interviews didn't go as in-depth as I hoped. I sometimes struggled to ask follow-up questions that would encourage deeper reflections. The interviews were all quite short which may be the reason why it didn't led into a deeper conversation.

What can be improved?

I've learned that refining my interview techniques, such as using more open-ended or probing follow-up questions. I think that can help draw out deeper responses. I also need to allocate more time for analysis and insight mapping in future sprints.

What actions will I take for the next sprint?

  • Start ideating solution concepts based on key opportunity areas
  • Create a future state user journey and create user flow
  • Start creating paper prototype
  • Improve time planning for synthesis and creative work
